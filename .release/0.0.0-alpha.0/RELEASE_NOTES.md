### Installation

Add to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/coinbase/cdp-swift", from: "0.0.0-alpha.0"),
]
```

Or add via Xcode: **File > Add Package Dependencies...** and enter `https://github.com/coinbase/cdp-swift`.
